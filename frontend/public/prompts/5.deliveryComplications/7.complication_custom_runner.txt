Custom deployment runner required.

Context:
CI/CD pipelines must run on a custom or self-hosted runner instead of managed runners.

Implications for CI/CD:
- Execution environment is not guaranteed to be preconfigured.
- Network and tool availability must be explicitly handled.

Requirements:
- Ensure required tools and dependencies are installed or validated at runtime.
- Account for network access to required systems (e.g., databases, APIs).
- Avoid assumptions about pre-installed software.
- Include setup or bootstrap steps if necessary.
